{{< alert tip >}}
PreBind の並列実行による Binding Cycle の高速化
{{< /alert >}}

* opt-in
* 並列化するプラグインは連続して配置されている必要がある  
    [serial] [parallel, parallel] [serial] [parallel]
    → [[p1], [p2, p3], [p4], [p5]]

```yaml {title="scheduler config"}
profiles:
  - plugins:
      multiPoints:
        enabled:
        ...
        - name: DRA
          allowParallel: true
        - name: volumebinding
          allowParallel: true
```
